# Desafio Técnico HCXpert — QA/SDET

Projeto inicial de automação Web e API utilizando Cypress, Cucumber/Gherkin e Page Object Model.

## Pré-requisitos

- Node.js 20, 22 ou 24
- npm
- Git
- Google Chrome atualizado

## Instalação

```bash
npm install
npx cypress verify
```

## Execução

```bash
npm run cy:open
npm run cy:run
```
